<static> getWebSocketFactory() → {WebSocketFactory}

Returns WebSocketFactory instance that is used to create connection if Kaazing's WebSocket implementation is used. This method returns a null if the browser's WebSocket implementation is being used. The WebSocketFactory instance can be used to set WebSocket related characteristics such as connection-timeout, challenge handlers, etc.
